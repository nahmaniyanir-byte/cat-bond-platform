# Global Sovereign Catastrophe Bond Explorer

Institutional policy intelligence platform for sovereign catastrophe bonds and disaster risk financing.

## Run

```powershell
cd C:\Users\nahma\PyCharmMiscProject\mmn.14\catbond_project\project-root
npm install
npm run dev
```

Open: `http://localhost:3000`

## Source of Truth

The app is dynamically connected to:

- `content/metadata/global.json`
- `content/metadata/countries.json`
- `content/global/...`
- `content/countries/...`
- `data/master/cat_bond_master_database.json` (generated from CSV)
- `data/countries/country_globe_points.json` (generated from CSV)
- `data/chart_data/*.json` (generated analytics datasets)

No country case data is hardcoded in page components.

## CSV Data Pipeline

Master input CSV:

- `C:\Users\nahma\PyCharmMiscProject\mmn.14\catbond_project\catbond_clean_sql_metabase_policy_package\deals_master_clean.csv`

Generate / refresh all datasets:

```powershell
cd C:\Users\nahma\PyCharmMiscProject\mmn.14\catbond_project\project-root
npm run data:refresh -- --csv "C:\Users\nahma\PyCharmMiscProject\mmn.14\catbond_project\catbond_clean_sql_metabase_policy_package\deals_master_clean.csv"
```

Outputs generated by the pipeline:

- `data/master/cat_bond_master_database.json`
- `data/master/country_globe_points_master.json`
- `data/master/country_page_index_master.json`
- `data/master/country_kpis_master.json`
- `data/chart_data/issuance_by_year.json`
- `data/chart_data/peril_distribution.json`
- `data/chart_data/spread_vs_expected_loss.json`
- `data/chart_data/sovereign_vs_private.json`
- `data/sovereign_deals/sovereign_deals_master.json`
- `data/private_deals/private_deals_master.json`
- `content/home/kpis.json` (auto-refreshed with real KPI values)

## Country Excel Pipeline

Country intelligence source root:

- `project-root/data/countries/{country}/*.xlsx`

Generate per-country JSON bundles and dynamic country page index:

```powershell
cd C:\Users\nahma\PyCharmMiscProject\mmn.14\catbond_project\project-root
npm run countries:refresh
```

Generated outputs include:

- `data/countries/{country}/json/country_overview.json`
