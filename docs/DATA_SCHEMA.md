# Data Schema — cat_bond_master.csv

Canonical dataset for the Global Catastrophe Bond Intelligence Platform.
Located at `data/master/cat_bond_master.csv`.

Generated by `scripts/scraper_artemis.py` — see [`scripts/README.md`](../scripts/README.md).

---

## Column Reference

| Column | Type | Description | Example |
|--------|------|-------------|---------|
| `deal_id` | string | MD5 hash of name+year, first 12 chars | `a3f9b2c1d4e5` |
| `deal_name` | string | Official transaction name from Artemis | `Sanders Re II Ltd. (Series 2025-2)` |
| `sponsor` | string | Cedant / protection buyer | `Allstate` |
| `issuer_spv` | string | Special purpose vehicle name | `Sanders Re II Ltd.` |
| `issue_year` | integer | Year of issuance | `2025` |
| `maturity_year` | integer | Expected maturity year | `2028` |
| `deal_size_usd` | float | Size in **USD millions** | `150.0` |
| `original_currency` | string | ISO currency code of original issuance | `USD` |
| `size_as_displayed` | string | Raw string verbatim from Artemis | `$150m` |
| `trigger_type` | string | Trigger architecture (see values below) | `Indemnity` |
| `peril` | string | Covered hazard(s) | `Hurricane/Wind` |
| `region` | string | Broad geographic region | `North America` |
| `country_of_risk` | string | Country/territory of covered risk | `United States` |
| `country_of_sponsor` | string | Sponsor home country | `United States` |
| `sovereign_flag` | boolean | True if government or multilateral is sponsor/beneficiary | `False` |
| `rating` | string | S&P, Moody's, or Fitch rating at issuance | `B+` |
| `expected_loss` | float | Expected loss as **percentage** (e.g. 1.74 = 1.74%) | `1.74` |
| `spread_bps` | float | Spread / coupon in **basis points** | `700.0` |
| `coupon_pct` | float | Coupon as percentage (spread_bps / 100) | `7.0` |
| `risk_multiple` | float | spread_bps / (expected_loss × 100) | `4.02` |
| `triggered` | boolean | Whether the bond was triggered (partial or total loss) | `False` |
| `principal_loss_pct` | float | Principal loss percentage if triggered | `null` |
| `source_url` | string | Artemis deal page URL | `https://www.artemis.bm/deal-directory/...` |
| `data_quality_score` | integer | Field completeness score 1–5 | `4` |
| `scraped_at` | datetime | ISO 8601 UTC timestamp of scrape | `2026-03-17T23:00:00Z` |

---

## Canonical Values

### trigger_type
| Value | Description |
|-------|-------------|
| `Indemnity` | Triggered by actual sponsor losses |
| `Parametric` | Triggered by physical measurements (wind speed, shake intensity) |
| `Industry Loss` | Triggered by industry-wide insured loss index |
| `Modelled Loss` | Triggered by modelled loss output |
| `Hybrid` | Combination of two or more trigger types |

### peril (canonical)
| Value | Covers |
|-------|--------|
| `Hurricane/Wind` | Named storms, typhoons, cyclones, wind |
| `Earthquake` | Seismic, fire following, tsunami |
| `Flood` | River flood, storm surge, pluvial |
| `Multi-Peril` | Two or more perils in a single transaction |
| `Wildfire` | Wildland-urban interface fire |
| `Mortality` | Excess mortality (pandemic, mass casualty) |
| `Other` | Drought, hail, terrorism, etc. |

## Units Summary

| Field | Unit | Notes |
|-------|------|-------|
| `deal_size_usd` | USD millions | EUR/JPY/GBP converted at contemporaneous FX |
| `expected_loss` | percentage points | e.g. `1.74` means 1.74% |
| `spread_bps` | basis points | e.g. `700` means 7.00% |
| `coupon_pct` | percentage | e.g. `7.0` means 7.00% |
| `risk_multiple` | ratio | e.g. `4.02` = 402 bps spread / 100 × 1.74% EL |

---

## Known Limitations

- **EL for multi-tranche deals** is extracted from the first tranche mentioned in body text
- **Spread for complex deals** (zero-coupon, at-discount tranches) may be inaccurate
- **Pre-2005 deals** often have fewer structured fields on Artemis pages
- `country_of_risk` and `country_of_sponsor` are inferred from text, not always explicit
- `triggered` flag is set heuristically from page text; manual review recommended for loss events
